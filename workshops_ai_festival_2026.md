## Oficinas Confirmadas

### 1. Sua Empresa Tem Chefes Demais
> Entenda como a IA está desmontando hierarquias artificiais e revelando quem realmente cria valor nas organizações.

**Por que importa:** A IA expõe estruturas organizacionais que apenas distribuem aprovação sem gerar valor. Workshop estratégico para quem lidera equipes ou quer entender o impacto da IA na gestão.

---

### 2. Promptar é o Novo Codar
> Como o valor do desenvolvedor migrou da escrita de códigos para a capacidade de pensar, especificar e orquestrar sistemas com IA.

**Por que importa:** Para engenheiros e desenvolvedores que querem entender onde está o novo diferencial competitivo. Diretamente conectado ao trabalho do RochaDev — o que ensinar e como posicionar programação na era dos agentes.

---

### 3. Existem Muitas IAs e Você Está Usando as Erradas
> Aprenda a escolher, combinar e arquitetar modelos de IA de forma inteligente.

**Por que importa:** Workshop prático de arquitetura multi-modelo — Claude, Gemini, GPT-4o, Llama, Gemma. Fundamental para quem constrói soluções verticais e precisa otimizar custo, qualidade e latência.

---

### 4. AI Self Defense
> As técnicas de defesa pessoal contra ataques de IA.

**Por que importa:** Prompt injection, deepfakes, desinformação gerada por IA, engenharia social automatizada. Essencial para quem lida com clientes corporativos e dados sensíveis (obras, contratos, saneamento).

---

### 5. Claude Cowork: IA como Colega de Trabalho ⭐
> Casos de uso enterprise e como times estão usando Cowork no dia a dia.

**Facilitador:** Henrique Savelli — Arquiteto de AI na Anthropic  

**Por que importa:** Workshop ministrado diretamente pelo representante da Anthropic no Brasil. Cowork é o agente de desktop da Anthropic integrado a GSuite, Slack, Figma e arquivos locais via MCP. Oportunidade única de ver o estado da arte em agentes agentic na prática.

---

### 6. Código Virou Commodity?
> Escrever código deixou de ser diferencial? Onde está o novo valor da engenharia de software na era da IA.

**Por que importa:** Questão central para o RochaDev: o que ensinar quando o código pode ser gerado? Workshop que vai redefinir o currículo dos próximos treinamentos.

---

### 7. Treinar Pessoas como em 2010 é Antiético
> Um convite direto para repensar educação corporativa, aprendizagem contínua e o papel do treinamento em um mundo acelerado por IA.

**Por que importa:** Diretamente estratégico para o Código sem Fronteiras e futuras edições do RochaDev. Como redesenhar a experiência de aprendizado quando as ferramentas mudam a cada mês?

---

### 8. GEO, AIO ou AEO
> O fim do "busca por palavras-chave": as dicas práticas para os agentes de IA recomendarem o seu negócio.

**Por que importa:** GEO (Generative Engine Optimization), AIO (Agent Intent Optimization — termo de Paul Accornero) e AEO (Answer Engine Optimization) são o novo SEO. Fundamental para visibilidade do RochaDev e dos produtos desenvolvidos para clientes.

---

### 9. Líderes Não Letrados
> Detectando sinais de lideranças onde a "ficha ainda não caiu".

**Por que importa:** Workshop de diagnóstico organizacional — como identificar e trabalhar com líderes que ainda não entenderam a urgência da IA. Útil para quem vende transformação digital para construtoras e prefeituras.

---

### 10. Claude Code em Ação ⭐
> A jornada completa de quem quer começar a usar Claude Code de verdade. Do setup básico até workflows mais avançados com agentes.

**Facilitador:** Henrique Savelli — Arquiteto de AI na Anthropic  

**Por que importa:** Workshop hands-on sobre Claude Code — ferramenta de linha de comando agentic da Anthropic. Oportunidade direta de aprender com quem está mais próximo do produto. Alta prioridade para desenvolvedores.

---

### 11. Beyond Vibe Coding
> Chega de lero-lero: Vibe coding é para os fracos! Codar na raça é para os ultrapassados.

**Por que importa:** A provocação boa: nem geração cega de código por IA, nem programação manual linha por linha. Como encontrar o meio-termo entre engenharia real e produtividade com IA?

## Perguntas para Levar aos Workshops

### Para os workshops com Henrique Savelli (Claude Code / Claude Cowork):
- Para um projeto de agente vertical em construção civil (consulta a normas ABNT, extração de dados de planilhas SINAPI, geração de memoriais), qual a diferença prática entre usar **Claude Code em modo agentic** versus construir com a API diretamente?
- Como o **MCP** habilita que o Claude Cowork se conecte a sistemas proprietários legados (como ERPs de construtoras — Sienge, UAU, Mega) sem expor dados sensíveis?
- Qual o limite atual do Claude Code para projetos com **múltiplos arquivos interdependentes** e contexto técnico pesado (ex: projeto full-stack de painel de gestão de obras)?

### Para "Existem Muitas IAs e Você Está Usando as Erradas":
- Para soluções com requisitos de **residência de dados no Brasil** (LGPD + setor público), qual a combinação de modelos que você recomenda em 2026?
- Quando faz sentido usar um **modelo pequeno fine-tunado** (ex: Gemma 3 treinado em normas ABNT) versus um modelo grande generalista com RAG?

### Para "Treinar Pessoas como em 2010 é Antiético":
- Como redesenhar uma jornada de aprendizado de **desenvolvimento web para engenheiros** quando as ferramentas de IA mudam a cada 3 meses?
- Qual o papel de **projetos reais entregues durante o treinamento** versus conteúdo teórico, dado o ritmo atual de mudança?

### Para "Beyond Vibe Coding":
- Onde está a linha entre **vibe coding aceitável** (produtividade) e vibe coding perigoso (dívida técnica oculta)?
- Para um produto entregue a **cliente regulado** (prefeitura, concessionária de saneamento), que práticas de engenharia não se negocia mesmo com IA gerando 80% do código?
